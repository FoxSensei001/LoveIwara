Same-Origin Fallback

If the local page cannot fetch the API, open https://apiq.iwara.tv/tags?limit=1&page=0 in the same browser, clear the Cloudflare challenge, then paste the script below into DevTools Console.

  1. Open the API page in a tab and wait until it is no longer blocked.
  2. Open DevTools Console on that tab.
  3. Paste the generated script and press Enter.
  4. The browser will download the JSON snapshot automatically.
